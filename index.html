Game Engine
Equipe

Victor Ivis

Descrição

Uma game engine simples em C++ feita com SDL2. O mapa é carregado usando um arquivo xml que guarda as posições no formato csv. As colisões são lidadas pelo algorítmo AABB e existe a implementação da física para gravidade, atrito, movimentação na diagonal usando seno e cosseno e outros funcionalidades que estão em desenvolvimento. Existe uma versão compilada tanto para windows quanto para linux nas aba de tags.
