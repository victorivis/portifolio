Game Engine
Equipe

Victor Ivis

Descrição

Uma game engine simples em C++ feita com SDL2. O mapa é carregado usando um arquivo xml que guarda as posições no formato csv. As colisões são lidadas pelo algorítmo AABB e existe a implementação da física para gravidade, atrito, movimentação na diagonal usando seno e cosseno e outros funcionalidades que estão em desenvolvimento. Existe uma versão compilada tanto para windows quanto para linux na aba de tags.

Aplicativo para Controle de Gastos
Equipe

Cauã Pires, Daniel Lima, Victor Ivis

Descrição

Um aplicativo bem simples que permite guardar despesas e lucros para visualizar graficamente a situação financeira do usuário.

Atuei no backend fazendo a parte de armazenamento dos dados, que é bastante simples porque não utilizou qualquer banco de dados, mas apenas o próprio local storage. Também fiz toda a parte de geração dos gráficos usando a biblioteca Chart.js e ainda auxiliei um pouco nos pop-ups que foram feitos com a biblioteca SweetAlert2.
