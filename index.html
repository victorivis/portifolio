Game Framework


        

            
Team

            

                
Victor Ivis

            

        


        

            
Description

            

                
A simple game framework in C++ built with SDL2. The map is loaded using an XML file that stores positions in CSV format.
                Collisions are handled using the AABB algorithm, and physics are implemented for gravity, friction, diagonal movement 
                using sine and cosine, among other features currently in development. A compiled version is available for both 
                Windows and Linux under the tags section.

Lei de Zipf


        

            
Equipe

            

                
Victor Ivis

            

        


        

            
Descrição

            

                

                    Uma apresentação feita com o Jupyter Lab utilizando a biblioteca MatPlotLib do python para 
                    os gráficos da distribuição da lei de zipf em diversos livros como Alice no País das Maravilhas, 
                     Críton, Dom Casmurro e outros. Essa lei diz que a 2ª palavra mais frequente aparece metade das vezes
                     que a primeira, a 3ª aparece um terço da primeira e assim por diante. Esse conceito tem 
                     diversas aplicações na área da estatística como detecção de plágios, processamento de linguagem natural e
                     até previsão do comportamento de populações.

Zipf's Law


        

            
Team

            

                
Victor Ivis

            

        


        

            
Description

            

                
A presentation created with Jupyter Lab using Python's MatPlotLib library to display Zipf's law 
                distribution in various books such as Alice in Wonderland, Crito, Dom Casmurro, and others. 
                This law states that the second most frequent word appears half as often as the first, the third 
                appears one third as often, and so on. This concept has many applications in statistics, such as 
                plagiarism detection, natural language processing, and population behavior prediction.
